Replace comma(s) with semicolon(s)

FileMaker calculations use semicolons (;) to separate function parameters. This intention replaces comma separators with semicolons.

Usage:

  • Place the caret on a comma within a .fmcalc file and invoke the intention to replace it with a semicolon.
  • Select a range containing commas to replace all commas in the selection at once.
  • Example: If ( test, resultTrue, resultFalse ) becomes If ( test; resultTrue; resultFalse ).
